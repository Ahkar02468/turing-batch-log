## Sort

* Array.sort() sort this array [10,23,110,13] to [10,13,110,23]. It sort lexicoscopically. Mans sort with string.
* If the arry is wated to sort with number, give a comparator parameter to sort() method.

```js
     sort((a,b) => a-b);
```
